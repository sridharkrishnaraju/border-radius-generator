# Usage & API

It is a standard custom element, so it works with no wrapper in plain HTML, React, Vue, Svelte and Astro.

## Plain HTML

```html
<script src="border-radius-tool.js"></script>
<border-radius-tool></border-radius-tool>
```

## React

```jsx
import "@sgbp/border-radius-tool";
export default function Page() { return <border-radius-tool />; }
```

## Vue

```vue
<script setup>
import "@sgbp/border-radius-tool";
</script>

<template>
  <border-radius-tool />
</template>
```
