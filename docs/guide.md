# Guide: Pick a radius scale and stick to it

Consistent rounding makes an interface feel considered. Choose a small scale — for example 8px for inputs and 14px for cards — and reuse it everywhere. For pill-shaped buttons, use a very large value like `999px`. Asymmetric radii (different per corner) are great for decorative shapes but use them sparingly.
