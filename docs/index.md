# Border Radius Generator

Round each corner with sliders, preview live, and copy the CSS. A free, zero-dependency Web Component you can drop into any site.

```html
<script src="border-radius-tool.js"></script>
<border-radius-tool></border-radius-tool>
```

## What it does

The CSS `border-radius` property rounds the corners of an element. You can set one value for every corner or a different radius per corner (top-left, top-right, bottom-right, bottom-left). This generator gives you a slider per corner, a live preview, and copy-ready CSS.

## When you'd use it

Dialling in the corner rounding for buttons, cards and inputs, or making asymmetric shapes like speech bubbles.

See the [Usage & API guide](usage.md) for framework examples, or the [guide](guide.md) for tips.
